
### 概要

ドラクエ4のカジノ（DS版）において、ポーカーでの獲得コイン枚数の期待値計算機です。

* 掛けているコインの枚数と引いたカードを元に、どのカードを残すと確率的に最も損をしないかを計算できます。

### 使い方

* ```src/jp/mayuge/MainActivity.java```の21行目、24-28行目を修正してください。
  * 21行目：掛けているコインの枚数（整数で入力）
  * 24-28行目：引いたカード
    * 関数```getCardId()```の第1引数にカードの数字(1-10, J:11, Q:12, K:13)、第2引数にマーク（```DQPoker.CARD_MARK_ID_****```）

* 修正後、javaが動作可能な環境で実行すると結果が出力されます。

### 特徴

* カードの重複があった場合に、誤修正として検知できます。
* 中間結果として、各カードを残した場合の獲得コイン期待値も表示されます。
* 中間結果、総合結果ともに、掛けているコインの枚数に対する獲得コインの損益枚数も計算します。
* 一部の計算パターン（4枚を交換、5枚を交換する場合）を省略する事で、処理の高速化を実現しました。
